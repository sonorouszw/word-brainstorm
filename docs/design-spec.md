# 设计规范 — 词语头脑风暴

## 设计理念

简洁、温暖、专注。让使用者专注于词语联想本身，不被复杂的界面干扰。

## 色彩方案

以暖色调为主，营造舒适、创意激发的感觉：

- 背景：米白 `#FFFBF5`
- 主色：暖橙 `#E07B39`
- 辅色：深棕 `#4A3728`
- 卡片背景：白 `#FFFFFF`
- 卡片边框：浅灰 `#E8E0D5`
- AI 模式标识：蓝紫 `#6C5CE7`
- 本地模式标识：翠绿 `#00B894`
- 选中高亮：暖金 `#FDCB6E`

## 字体

- 主字体：系统默认中文字体栈（PingFang SC, Microsoft YaHei, sans-serif）
- 词语展示：加粗，1.5 倍字号
- 拼音：小号，浅色
- 释义：常规字重，灰色

## 布局

### 桌面端
```
┌─────────────────────────────────────┐
│         词语头脑风暴 (标题)           │
│  [离线词典] [AI大模型]   (模式切换)    │
│  [________________] 输入框           │
│  "输入一个词来进行头脑风暴吧"          │
├─────────────────────────────────────┤
│                                     │
│    [根词]                           │
│    ├── [关联词1] [关联词2] ...      │
│    │    ├── [子关联词...]           │
│    │                               │
│    (词树区域，可滚动)                │
│                                     │
├─────────────────────────────────────┤
│  已选: [词A] [词B]  [合并联想 →]    │  (合并面板，选中2+词时出现)
└─────────────────────────────────────┘
```

### 移动端
- 词树垂直排列（非横向）
- 合并面板固定在底部
- 输入框置顶

## 组件规范

### WordInput
- 输入框：大号圆角输入框，居中
- 提示文字：浅灰色，14px
- 提交按钮：暖橙色，圆角

### WordNode（词语卡片）
- 白色卡片，圆角 8px，阴影
- 词语：18px 加粗
- 拼音：12px 浅色
- 释义：14px 灰色，可折叠
- 点击展开后出现子词语
- 选中状态：金色边框

### SourceSelector（模式切换）
- 两个大卡片并排
- 选中状态有边框高亮
- 每张卡片内列 2-3 条优点 + 1-2 条缺点
- AI 模式下额外显示 API 提供商选择

### BrainstormCanvas（词树画布）
- 从根词开始，横向展开
- 支持滚动和缩放（后期）
- 父词到子词有连线（Phase 4）

### MergePanel（合并面板）
- 固定在底部或侧边
- 显示已选词语标签
- 合并按钮，少于 2 个选中时禁用
- 合并结果弹出展示

## 交互

- 点击词语 → 展开/收起子词语
- Ctrl+点击 → 选中/取消选中词语（用于合并）
- 输入框回车 → 提交新词（重置词树）
- 模式切换 → 保留当前词树，新扩展使用新模式
