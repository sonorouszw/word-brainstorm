# 需求文档 — 词语头脑风暴

## 项目目标

帮助使用者将头脑中模糊的词语或想法逐步清晰化，通过「输入词 → 扩展关联词 → 继续扩展 → 连接合并」的流程，最终找到自己真正想表达的词语或概念。

## 核心功能

### 1. 输入词语
- 首界面有一个输入框
- 提示文字：「输入一个词来进行头脑风暴吧」
- 支持键盘回车提交
- 语音输入（首版不做，后续迭代）

### 2. 生成关联词
提供两种模式，用户可自由切换：

**模式一：离线词典**
- 基于本地中文词典（CC-CEDICT）
- 通过共享汉字匹配找到关联词
- 即时响应、无需网络
- 词汇量约 2 万常用词

**模式二：AI 大模型**
- 调用 Claude / OpenAI API
- 深度语义联想
- 需要 API Key 和网络
- 结果更丰富、更有创意

每种模式旁边展示利弊说明。

### 3. 继续扩展
- 点击任意关联词，以它为中心继续生成下一层关联词
- 形成词语联想树
- 可展开/收起任意分支
- 限制深度 5 层，每层最多 20 个

### 4. 词语合并
- 可选择 2-5 个词语
- 点击「合并」生成与所有选中词相关的新内容
- 本地模式：找出所有选中词的共同关联词
- AI 模式：生成创意串联文本、共同主题、新关联词

## 非功能需求

- 首版运行在浏览器（网页版）
- 后期可迁移到微信小程序 / 手机 App
- 界面简洁、适合编程新手理解和修改
- 所有业务逻辑代码使用中文注释

## 暂不做

- 语音输入（后续版本）
- 用户账号系统
- 历史记录保存
- 多语言支持（仅中文）
